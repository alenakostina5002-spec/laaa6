Калькулятор трафика Интернет

Как это работает?

При вводе значения в поле "Килобиты" автоматически рассчитывается значение в килобайтах.

При получении фокуса полем "Килобайты" происходит пересчет значения на основе текущего значения в килобитах.

Формула конвертации: 1 килобайт (КБ) = 8 килобит (Кб)

Таким образом, чтобы перевести килобиты в килобайты, нужно разделить значение на 8.

Пример: 64 Кб = 64 / 8 = 8 КБ
